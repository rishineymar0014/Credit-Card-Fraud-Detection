# 💳 Credit Card Fraud Detection using Machine Learning

An AI-powered Machine Learning application that predicts whether a credit card transaction is **Legitimate** or **Fraudulent**.

## 🚀 Live Demo

🔗 Streamlit App:  
https://credit-card-fraud-detection-giiqmm5n7swqkubae7et7p.streamlit.app/


## 📌 Project Overview

Credit card fraud detection is a critical problem in the financial industry.

This project uses Machine Learning algorithms to analyze transaction patterns and identify suspicious transactions.

The application takes transaction details as input and predicts whether the transaction is safe or fraudulent with a confidence score.


## ✨ Features

✅ Fraud detection using Machine Learning  
✅ Legitimate vs Fraud transaction classification  
✅ Prediction probability/confidence score  
✅ Interactive web interface using Streamlit  
✅ Real-time fraud prediction

## 📊 Model Performance

The trained model was evaluated using classification metrics:

- Accuracy: **95.94%**
- Precision: **98%**
- Recall: **94%**
- F1 Score: **96%**
